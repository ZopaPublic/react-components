Use `<SiedContainer />` to set a `max-width` around its children:

#### Short size

```jsx
import { SizedContainer } from '@zopauk/react-components';

<SizedContainer size="short">
  <div style={{ backgroundColor: 'tomato', height: '100%' }}>This is a container</div>
</SizedContainer>;
```

#### Medium size

```jsx
import { SizedContainer } from '@zopauk/react-components';

  <div style={{ backgroundColor: 'tomato', height: '100%' }}>This is a container</div>
```
